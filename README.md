# 영어 문장 연습기 (English Sentence Practice)

영어 문장 연습부터 상황별 회화·회화 표현·단어 뜻까지 한 번에 익히는 학습용 웹 앱입니다.
빌드 과정이 없는 정적 파일이며, 설치형 PWA로 동작해 모바일/데스크톱에서 오프라인으로도 사용할 수 있습니다.

## 주요 기능

- 난이도별(초급·중급·고급) 주제 문장 연습 — 문장 속 형용사·동사를 바꿔 가며 패턴 학습
- 내 문장 읽기(커스텀) — 입력한 문장 음성 읽기(반복 가능) · 오프라인 단어 뜻 / 온라인 문장 해석
- 상황별 회화(스몰토크) — 두세 마디 대화, 답변 바꿔 보기
- 회화 표현 — 자주 쓰는 표현·이디엄의 뜻·사용 상황·예문
- 듣기 속도 5단계 · 여성/남성/아이 음성 선택 · 즐겨찾기
- PWA 지원 (홈 화면 설치 · 오프라인 캐시)

## 배포 / 실행

이 폴더 자체가 배포 가능한 정적 사이트입니다. 진입 파일은 `index.html`입니다.

- **정적 호스팅(권장):** 이 폴더의 파일을 Netlify Drop(app.netlify.com/drop)에 드래그하거나 GitHub Pages 등으로 올리면 됩니다.
- **로컬 미리보기:**

```bash
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000/ 접속
```

> PWA(서비스 워커·설치) 기능은 `file://` 직접 열기에서는 제한될 수 있으므로 호스팅 또는 로컬 서버에서 실행하세요.

## 구성 파일

| 파일 | 설명 |
|---|---|
| `index.html` | 앱 본체 (HTML/CSS/JS 단일 파일, 진입점) |
| `manifest.webmanifest` | PWA 매니페스트 |
| `sw.js` | 서비스 워커 (오프라인 캐시) |
| `icon-192.png`, `icon-512.png` | 앱 아이콘 |
